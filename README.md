
# Proxy Cache Demo 

Projekt zum Video:

https://youtu.be/7BgkR98-anA?si=yPuSaBrcZuLh9-XN

1. Starte proxy server:

```bash
./run-squid.sh
```

oder:

```bash
./run-nginx.sh
```

oder:

```bash
./run-varnish.sh
```

2. Führe die App-Klasse aus, um die Spring Boot Anwendung zu starten.

3. Führe die Anfragen aus dem Video aus.

# Client Cache Demo ETag
